8.
int sum(int x, int y)
{
	int result;
	result=x+y;
	return result;
}	
void main()
{
	printf("%d",sum(3,4));
}
위의 소스에서 각 부분을 지칭하는 용어를 작성하시오.
①int sum 에서 'int'가 있는 부분 : 정수
②int x,int y 가 있는 부분 : 파라미터
③sum(3,4) : 함수 -> 파라미터로 sum 함수에 값을 전달하여 계산한다.
④3,4가 있는 부분 : sum 함수로 전달하는 값
